Generate a conventional commit message. Output ONLY the commit message.

TEMPLATE: type(scope): description

RULES:
- Max 50 chars
- Imperative mood
- English only
- No punctuation at end

TYPES:
feat: new feature
fix: bug fix
docs: documentation
style: formatting
refactor: code restructure
test: testing
chore: maintenance
perf: performance
ci: continuous integration
build: build system

EXAMPLES:
- feat(auth): Add OAuth2 support
- fix(api): Handle null pointer
- docs: Update Install guide
- refactor: Simplify query logic
- chore: Update dependencies
